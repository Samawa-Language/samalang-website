## Mengapa Memilih SamaLang?

<CardGrid stagger>
	<Card title="Sintaks Lokal" icon="translate">

	<Card title="Performa Lua" icon="star">

	</Card>
	<Card title="Mudah Dipelajari" icon="rocket">

## Contoh Kode SamaLang

Lihat bagaimana SamaLang memadukan logika tradisional dengan performa modern.

<Tabs>
  <TabItem label="Hello World">

    ```bash
    # Sapaan tradisional
    tulis("Salam Sarea!")
    ```
  </TabItem>
  <TabItem label="Variabel Lokal">
    ```bash
    ada aran = "SamaLang"
    ada visi = "Digitalisasi Budaya"

    tulis(aran .. " v2.0")
    ```
  </TabItem>
  <TabItem label="Fungsi">

    ```bash
    jure_mo
    ```

    ```bash
    lamen umur >= 18 tres
        tulis("Dewasa")
    jure_mo
    ```
  </TabItem>

## Pemetaan Kata Kunci

SamaLang menggunakan kata kunci Samawa yang ditranspile langsung ke Lua:

| SamaLang | Lua | Kegunaan |
|----------|-----|----------|
| `ada` | `local` | Deklarasi variabel |
| `fungsi` | `function` | Definisi fungsi |
| `lamen` | `if` | Kondisional |
| `tres` | `then` | Awal blok |
| `jure_mo` | `end` | Akhir blok |
| `selama` | `while` | Perulangan while |
| `untuk` | `for` | Perulangan for |
| `tulis` | `print` | Cetak output |
| `semalik` | `return` | Kembalikan nilai |
| `nda_isi` | `nil` | Nilai kosong |
