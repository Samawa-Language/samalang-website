## Warning: Eksperimental

Ekstensi ini masih dalam tahap eksperimental.
Beberapa fitur mungkin tidak berfungsi seperti yang diharapkan dan masih dalam pengembangan aktif.

## Description / Deskripsi

Samalang menyediakan dukungan syntax highlighting untuk Zed menggunakan ekstensi kustom yang didukung oleh Tree-sitter.
Ini memungkinkan file `.sl` dikenali dengan highlighting yang proper dan struktur bahasa.

## Instalasi

### Instalasi Manual

Clone repository ekstensi dan ikuti instruksi build:

```bash
git clone https://github.com/Samawa-Language/samalang-zed-extension.git
```
