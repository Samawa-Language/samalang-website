## Contoh Kode Samalang

Lihat bagaimana Samalang memadukan logika tradisional dengan performa modern.

<Tabs>
  <TabItem label="Hello World">
    ```sl
    # Sapaan tradisional
    tulis("Salam Sarea!")
    ```
  </TabItem>
  <TabItem label="Variabel Lokal">
    ```sl
    ada aran = "Samalang"
    ada visi = "Digitalisasi Budaya"

    tulis(aran .. " v2.0")
    ```
  </TabItem>
  <TabItem label="Fungsi">
    ```sl
    fungsi sapa(nama)
        semalik "Halo, " .. nama .. "!"
    jure_mo

    tulis(sapa("Dunia"))
    ```
  </TabItem>
  <TabItem label="Percabangan">
    ```sl
    ada umur = 20

    lamen umur >= 18 tres
        tulis("Dewasa")
    jure_mo
    ```
  </TabItem>
</Tabs>

## Pemetaan Kata Kunci (21 Keywords)

Samalang v2.0 menggunakan 21 kata kunci Samawa yang ditranspile langsung ke Lua:

| Samalang | Lua | Kegunaan |
|----------|-----|----------|
| `ada` | `local` | Deklarasi variabel |
| `fungsi` | `function` | Definisi fungsi |
| `lamen` | `if` | Kondisional |
| `tres` | `then` | Awal blok |
| `lamen_no` | `else` | Alternatif kondisi |
| `lamen_no_kebali` | `elseif` | Kondisi tambahan |
| `jure_mo` | `end` | Akhir blok |
| `selama` | `while` | Perulangan while |
| `unti` | `while` (alt) | Perulangan while alternatif |
| `boat` | `do` | Awal blok perulangan |
| `untuk` | `for` | Perulangan for |
| `sampe` | `until` | Perulangan repeat |
| `jangka_mo` | `break` | Hentikan perulangan |
| `tulis` | `print` | Cetak output |
| `nda_isi` | `nil` | Nilai kosong |
| `no` | `not` | Logika NOT |
| `dan` | `and` | Logika AND |
| `atau` | `or` | Logika OR |
| `tutu` | `true` | Benar |
| `siong` | `false` | Salah |
| `semalik` | `return` | Kembalikan nilai |
