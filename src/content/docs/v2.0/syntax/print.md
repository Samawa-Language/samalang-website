## tulis (print)

`tulis` prints values to the console.

> **Archived:** This is documentation for Samalang v2.0. [View latest docs](/)

### Example

```sl
tulis("Halo Dunia!")
tulis(1 + 2)
tulis("Sama", "Lang")
```

### Lua Equivalent

```lua
print("Halo Dunia!")
print(1 + 2)
print("Sama", "Lang")
```
