## tutu (true)

`tutu` represents the boolean true value.

> **Archived:** This is documentation for Samalang v2.0. [View latest docs](/)

### Example

```sl
ada aktif = tutu

lamen aktif tres
    tulis("Active!")
jure_mo
```

### Lua Equivalent

```lua
local aktif = true

if aktif then
    print("Active!")
end
```
