## Quick Look at SamaLang v2.0

See how SamaLang v2.0 blends traditional logic with modern performance.

  <TabItem label="Local Variables">

## Keyword Mapping

SamaLang v2.0 uses **21 Samawa keywords** that transpile directly to Lua:

| SamaLang | Lua | Meaning |
|----------|-----|---------|
| `ada` | `local` | Declare variable |
| `fungsi` | `function` | Define function |
| `lamen` | `if` | Conditional |
| `tres` | `then` | Start block |
| `lamen_no` | `else` | Alternative branch |
| `lamen_no_kebali` | `elseif` | Additional conditional branch |
| `jure_mo` | `end` | End block |
| `selama` | `while` | While loop |
| `unti` | `while` | While loop (alternate) |
| `boat` | `do` | Start loop body |
| `untuk` | `for` | Numeric for loop |
| `sampe` | `until` | Repeat-until loop |
| `jangka_mo` | `break` | Exit loop |
| `tulis` | `print` | Print output |
| `nda_isi` | `nil` | Null value |
| `no` | `not` | Logical NOT |
| `dan` | `and` | Logical AND |
| `atau` | `or` | Logical OR |
| `tutu` | `true` | Boolean true |
| `siong` | `false` | Boolean false |
| `semalik` | `return` | Return value |

:::note
SamaLang v2.0 uses `#` for comments (not `--` like Lua).
:::

## What's New in v2.0

SamaLang v2.0 expanded from **11 to 21 keywords**, adding:

- **Boolean literals:** `tutu` (true) and `siong` (false)
- **Logical operators:** `no` (not), `dan` (and), `atau` (or)
- **Control flow:** `lamen_no_kebali` (elseif), `sampe` (until), `jangka_mo` (break)
- **Loop keywords:** `unti` (while alt), `boat` (do)
