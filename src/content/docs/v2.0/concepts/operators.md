## Operators

Samalang v2.0 supports standard Lua operators.

> **Archived:** This is documentation for Samalang v2.0. [View latest docs](/)

### Arithmetic

| Operator | Meaning |
|----------|---------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulo |
| `^` | Power |

### Comparison

| Operator | Meaning |
|----------|---------|
| `==` | Equal |
| `~=` | Not equal |
| `<` | Less than |
| `>` | Greater than |
| `<=` | Less or equal |
| `>=` | Greater or equal |

### Logical

| Samalang | Lua | Meaning |
|----------|-----|---------|
| `dan` | `and` | Logical AND |
| `atau` | `or` | Logical OR |
| `no` | `not` | Logical NOT |

### String

| Operator | Meaning |
|----------|---------|
| `..` | Concatenation |
