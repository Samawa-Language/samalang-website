## Data Types

Samalang v2.0 supports the following data types, inherited from Lua.

> **Archived:** This is documentation for Samalang v2.0. [View latest docs](/)

### Number

```sl
ada x = 42
ada pi = 3.14
```

### String

```sl
ada nama = "Samalang"
ada greeting = 'Hello'
```

### Boolean

```sl
ada aktif = tutu
ada mati = siong
```

### Nil

```sl
ada x = nda_isi
```

### Table

```sl
ada mahasiswa = { "Rudi", "Sari", "Budi" }
tulis(mahasiswa[1])
```
