## How to Install Samalang?
Choose the installation method that best suits your needs. The fastest way is using NPM, but you can also perform a manual installation by cloning directly from our official repository. Make sure you have Node.js installed on your system before starting the installation process via NPM.

#### Install Compiler use Npm (Recommended)

```bash
npm i -g samalang-compiler
````
#### Copy from repository

```bash
git clone https://github.com/Samawa-Language/samawa-language.git
```
