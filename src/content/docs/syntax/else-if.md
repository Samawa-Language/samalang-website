## Description
The `lamen_no_kebali` keyword in Samalang provides an else-if branch.  
It is equivalent to `elseif` in standard Lua.

## Usage
```bash
lamen condition1 tres
    -- code
lamen_no_kebali condition2 tres
    -- code
lamen_no
    -- code
jure_mo
```
