## Quick Look at SamaLang v1.0

See how SamaLang v1.0 blends traditional logic with modern performance.

<Tabs>
  <TabItem label="Hello World">
    ```sl
    # Traditional greeting
    tulis("Salam Sarea!")
    ```
  </TabItem>
  <TabItem label="Local Variables">
    ```sl
    ada aran = "SamaLang"
    ada visi = "Digitalizing Culture"

    tulis(aran .. " v1.0")
    tulis("Visi: " .. visi)
    ```
  </TabItem>
  <TabItem label="Function">
    ```sl
    fungsi sapa(nama)
        semalik "Halo, " .. nama .. "!"
    jure_mo

    tulis(sapa("Dunia"))
    ```
  </TabItem>
  <TabItem label="Control Flow">
    ```sl
    ada umur = 20

    lamen umur >= 18 tres
        tulis("Dewasa")
    jure_mo
    lamen_no
        tulis("Anak-anak")
    jure_mo
    ```
  </TabItem>
  <TabItem label="Loop">
    ```sl
    # Countdown
    untuk i = 5, 1, -1 boat
        tulis(i)
    jure_mo
    tulis("Lepas landas!")
    ```
  </TabItem>

## Keyword Mapping

SamaLang v1.0 uses **11 Samawa keywords** that transpile directly to Lua:

| SamaLang | Lua | Meaning |
|----------|-----|---------|
| `ada` | `local` | Declare variable |
| `fungsi` | `function` | Define function |
| `lamen` | `if` | Conditional |
| `tres` | `then` | Start block |
| `lamen_no` | `else` | Alternative branch |
| `jure_mo` | `end` | End block |
| `selama` | `while` | While loop |
| `untuk` | `for` | For loop |
| `tulis` | `print` | Print output |
| `semalik` | `return` | Return value |
| `nda_isi` | `nil` | Null value |

:::note
SamaLang v1.0 uses `#` for comments (not `--` like Lua).
:::
