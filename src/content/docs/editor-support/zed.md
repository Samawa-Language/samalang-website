## ⚠️ Experimental
This extension is currently in an experimental stage.  
Some features may not work as expected and are still under active development.

## Description
Samalang provides syntax highlighting support for Zed using a custom extension powered by Tree-sitter.  
This enables `.sl` files to be recognized with proper highlighting and language structure.
