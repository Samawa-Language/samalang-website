## Quick Look at SamaLang

See how SamaLang blends traditional logic with modern performance.

<Tabs>
  <TabItem label="Hello World">
    ```bash
    # Sapaan tradisional
    tulis("Salam Sarea!")
    ```
  </TabItem>
  <TabItem label="Local Variables">
    ```bash
    ada aran = "SamaLang"
    ada visi = "Digitalizing Culture"

    tulis(aran .. " v2.1")
    tulis("Visi: " .. visi)
    ```
  </TabItem>
  <TabItem label="Function">
    ```bash
    fungsi sapa(nama)
        semalik "Halo, " .. nama .. "!"
    jure_mo

    tulis(sapa("Dunia"))
    ```
  </TabItem>
  <TabItem label="Control Flow">
    ```bash
    ada umur = 20

    lamen umur >= 18 tres
        tulis("Dewasa")
    jure_mo
    lamen_no
        tulis("Anak-anak")
    jure_mo
    ```
  </TabItem>
  <TabItem label="Loop">
    ```bash
    # Hitung mundur
    untuk i = 5, 1, -1 boat
        tulis(i)
    jure_mo
    tulis("Lepas landas!")
    ```
  </TabItem>

## Keyword Mapping

SamaLang uses Samawa keywords that transpile directly to Lua:

| SamaLang | Lua | Meaning |
|----------|-----|---------|
| `ada` | `local` | Declare variable |
| `fungsi` | `function` | Define function |
| `lamen` | `if` | Conditional |
| `tres` | `then` | Start block |
| `jure_mo` | `end` | End block |
| `selama` | `while` | While loop |
| `untuk` | `for` | For loop |
| `tulis` | `print` | Print output |
| `semalik` | `return` | Return value |
| `nda_isi` | `nil` | Null value |
